# Using WhatFont Extension

You can use the WhatFont extension provided by Google Chrome to inspect the fonts used in a web page.

## Step 1: Get WhatFont extension
1.	Go to the Chrome Web Store (https://chromewebstore.google.com).
2.	Search to locate the WhatFont extension.
3.	Click the Get/Add to Chrome button.
4.	In the window that is displayed, click **Add Extension**.

## Step 2: Use WhatFont extension
1.	Navigate to any website. 
2.	In the top-right of the browser, click the Extensions icon (puzzle icon).
3.	Hover over any text to view the font of the text.
4.	Click the text to view other details of the font such as Family, Style, Weight, Color, Size, and Line Color.
5.	Press Esc to deactivate WhatFont.
